# [View resume](https://github.com/Bumvolla/Resume/blob/main/cv.pdf)

## PDF compilation

1. Get MiKTex.
2. Get and install the Fontin font from http://www.exljbris.com/fontin.html.
3. Open a cmd and navigate to the repository folder.
4. Launch:
~~~
xelatex cv.tex
~~~
